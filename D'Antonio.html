What is the class used to create a column that takes up half the page?

.col-6

What is the class used to contain all of the elements on the page?

.container

What is the class used to create a yellow button?

.btn-warning

What is the class used to create a white button with a green outline?

.btn-outline-success

What is one of the classes used to position columns horizontally on the screen when they don't fill up the 12 default columns?

.justify-content-between

What is another one of the classes used to position columns horizontally on the screen when they don't fill up the 12 default columns?

.justify-content-end

What is the class called when you want to add margin to the top(and then you want to add a dash plus a number 1-5)?

.margin-top-5

What is the class called when you want to add padding to the top AND bottom(and then you want to add a dash plus a number 1-5)?

.padding-top-3 .padding-bottom-3
